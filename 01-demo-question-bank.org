This file demonstrates question bank export with =ox-canvas-quiz.el=.
Export with ~C-c C-e Q b~ (=.md=) or ~C-c C-e Q B~ (=.zip=).

The =#+TITLE:= becomes the question bank name in Canvas.  After
importing the =.zip=, create a quiz in Canvas and add question groups
that pull from this bank.

Unlike quiz export, question bank export:
- Uses =Question Bank Title:= instead of =Quiz Title:=
- Produces all questions as standalone (no GROUP blocks)
- Ignores =#+QUIZ_SHUFFLE:= (shuffle is configured per-quiz in Canvas)

See =00-demo-all-question-types.org= for a quiz export demo.
